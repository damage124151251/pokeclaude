AI plays Pokemon Fire Red in real-time using Claude Vision.

## Features

- Real-time game screen visualization
- Pokemon team tracking with stats, HP, moves
- Battle log with AI strategy explanations
- Badge progress tracking
- Kanto map with current location
- Live terminal with AI decision-making process

## Tech Stack

- Next.js 14
- Supabase (Real-time database)
- Tailwind CSS (Retro Game Boy theme)
- Claude AI (Vision API)
- mGBA Emulator

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```
